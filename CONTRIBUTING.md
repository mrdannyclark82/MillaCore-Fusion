# Contributing

- Use pnpm workspaces to install and run apps: pnpm install
- Use pnpm dev:milla-gem or pnpm dev:gemini-assistant to start frontends
- To run both frontends behind a proxy: docker-compose -f docker-compose.dev.yml up
- When performing deduplication: run scripts/dedupe.py and consolidate code into packages/common or packages/ai-core
